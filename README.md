## 添加至本地
**使用 Termux**
```
termux-setup-storage
cd ~/storage/downloads
git clone https://github.com/forliuyifei/Configs.git
cd
ln -s /storage/emulated/0/Download/Configs/ Configs
```
*或者 `git clone git@github.com:forliuyifei/Configs.git`*
